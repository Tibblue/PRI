Resumo

Este trabalho contém um DTD capaz de verificar que a estrutura de uma página CLAV está a ser seguida corretamente, mantendo a informação consistente e partilhável entre várias pessoas.

Este trabalho contém também um ficheiro XML, um XSLT e o HTML gerado apartir de ambos. Utilizando a facilidade de processamento de um XML e as capacidades do XSLT, geramos um HTML que facilmente mostra a informação pretendida

Extra:

O XSL criado verifica se uma tag opcional não tem conteúdo, e caso se verifique, a tag é ignorada e não apresentada. Isto evita casos em que a tag esteja presente no XML, mas vazia e o DTD não tem motivos para dar erro, pois a tag era opcional.

Resultados

  • DTD para o processo de negócio
  • XML do manifesto
  • XSL de comversão do XML para HTML
  • HTML resultado
