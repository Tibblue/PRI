Resumo

Este trabalho pretendia criar um website que permitisse o upload de ficheiros por um utilizador, e que os mesmo fossem depois acessiveis ao mesmo.

Para tal usamos um servidor em NodeJS, que respondia ao cliente com a página contendo o formulário para upload do ficheiro e tambem uma listagem dos ficheiros já esistentes. O ficheiros ficheiros.json serve como base de dados.

Usa-mos tambem um script em JavaScript que permitia a utilização de 2 botões na página para esconder/visualizar a lista de ficheiros.
