Resumo

Este trabalho pretendia criar um website que permitisse registo, comfirmação do mesmo e listagem de teses.

Para tal usamos um servidor em NodeJS, que respondia ao cliente com a página pedida. Para tal este utiliza o ficheiro teses.json como base de dados.

Usa tambem várias .pug que são responsaveis pela formatação das páginas web.

Resultados

  • GitHub Repo TPC6
  • XML do manifesto
  • HTML do manifesto (ficheiro atual)
  • XSL do manifesto
  • Pasta com os .pug responsaveis pela formatação das páginas
  • Servidor em NodeJS
  • Ficheiro JSON que armazena a lista de teses
